# Balinese Cultural AI Dataset

## Description
This dataset contains structured cultural knowledge from Bali, Indonesia.
It is designed for research in Artificial Intelligence, Multimodal Learning,
and Creative & Cultural Tourism.

## Contents
- Cultural dances
- Ritual descriptions
- Cultural philosophy
- Associated visual samples

## Use Case
- AI chatbot for cultural tourism
- Multimodal AI research
- Educational systems
- Advanced Artificial Intelligence coursework

## Image Citation

All images in this dataset are AI-generated illustrative images created
for educational and research purposes.

Image generation prompts are designed to represent Balinese cultural
elements accurately but do not replace authentic ethnographic
documentation.

Generated using:
- Text-to-Image Generative AI (Stable Diffusion / SDXL class models)
- Prompt-based cultural description
